# Student Management System (Java CRUD)

## Description
This project is a simple Java-based Student Management System that demonstrates CRUD (Create, Read, Update, Delete) operations using Object-Oriented Programming concepts. It allows users to manage student records through a console-based interface.

## Features
- Add a new student
- View all student records
- Update existing student details
- Delete student records

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections (ArrayList)

## Project Structure
- `Student.java` – Student model class
- `StudentService.java` – Handles CRUD operations
- `Main.java` – Runs the application and displays the menu

## Learning Objectives
- Understand Java classes and objects
- Practice CRUD operations
- Work with ArrayList collections
- Implement menu-driven applications
- Apply Object-Oriented Programming principles

## Future Enhancements
- Store data in a database (MySQL)
- Add file handling for data persistence
- Implement validation and exception handling
- Build a GUI using Java Swing or JavaFX
